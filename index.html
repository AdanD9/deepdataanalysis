Breast Cancer Classification Using Neural Networks

A comprehensive machine learning approach to medical diagnosis

Project Overview

This project demonstrates the application of neural networks for medical diagnosis, specifically classifying breast cancer tumors as malignant or benign. Using the Wisconsin Breast Cancer Dataset, I developed a robust classification model that achieved exceptional performance with minimal false negatives - a critical requirement in medical applications.

Key Achievement: Developed a neural network model with 98.2% accuracy and only 2 false negative cases out of 114 test samples, demonstrating excellent reliability for medical diagnosis applications.

Dataset Description

The Wisconsin Breast Cancer Dataset contains 569 samples with 30 numerical features computed from digitized images of breast mass cell nuclei. Each sample represents measurements of cell characteristics including:

  • Radius: Mean distance from center to perimeter points
  • Texture: Standard deviation of gray-scale values
  • Perimeter: Tumor boundary length
  • Area: Tumor area measurement
  • Smoothness: Local variation in radius lengths
  • Compactness: PerimeterÂ² / area - 1.0
  • Concavity: Severity of concave portions
  • Concave Points: Number of concave boundary portions
  • Symmetry: Tumor symmetry measurement
  • Fractal Dimension: "Coastline approximation" - 1

Each feature includes mean, standard error, and "worst" (largest) values, resulting in 30 total features per sample.

Model Architecture

Implemented a Multi-Layer Perceptron (MLP) with the following architecture:

  • Input Layer: 30 features (all tumor characteristics)
  • Hidden Layer 1: 100 neurons with ReLU activation
  • Hidden Layer 2: 50 neurons with ReLU activation
  • Output Layer: 1 neuron with sigmoid activation (binary classification)
  • Optimizer: Adam optimizer with adaptive learning rate
  • Max Iterations: 1000 (with early stopping)
